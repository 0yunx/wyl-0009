# 🚀 躲避陨石 (Meteor Dodge)

一个使用 HTML5 Canvas + CSS3 + 原生 JavaScript 实现的休闲小游戏，单文件零依赖，打开即玩。

---

## 游戏玩法

- **目标**：操控飞船躲避从天而降的陨石，存活时间越长，分数越高。
- **难度递增**：随着时间推进，陨石下落速度会越来越快，生成间隔也会缩短，挑战持续升级。
- **碰撞即结束**：飞船与任意陨石发生碰撞后游戏立即结束。
- **最高纪录**：自动使用浏览器 `localStorage` 持久保存你的历史最高分。

---

## 操作方式

| 操作 | 按键 |
|------|------|
| 向左移动 | `←` 方向键 或 `A` 键 |
| 向右移动 | `→` 方向键 或 `D` 键 |
| 暂停 / 继续 | `空格键` |
| 重新开始 | `R` 键 或 `Enter` 键 |

也可以点击界面下方的按钮进行操作：**开始游戏**、**暂停 / 继续**、**重新开始**。

---

## 运行方式

### 方法一：直接打开（最简单）

直接双击 `index.html` 文件，或右键选择「打开方式」→ 任意现代浏览器（Chrome、Edge、Firefox、Safari 等）即可开始游戏。

### 方法二：本地 HTTP 服务器（可选）

某些浏览器对本地文件的某些特性有限制时，可以启动一个简单的本地服务器：

**使用 Python 3：**
```bash
python -m http.server 8000
```

**使用 Node.js（需全局安装 serve）：**
```bash
npx serve .
```

启动后在浏览器访问 `http://localhost:8000`（或对应端口）即可。

## 技术说明

- **HTML5 Canvas**：负责游戏画面的渲染（飞船、陨石、星空背景等）。
- **原生 JavaScript**：处理游戏循环、输入监听、碰撞检测、状态管理。
- **CSS3**：负责界面布局、渐变背景、按钮动效等视觉样式。
- **localStorage**：持久化保存最高分数。
- **单文件结构**：所有 HTML、CSS、JavaScript 代码全部内联在 `index.html` 中，无需构建工具与任何第三方依赖。
